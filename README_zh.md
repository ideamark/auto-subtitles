# Subtitle Generator

## 综述

Subtitle generator 是一款自动电影字幕生成工具。

## 安装

* 下载安装 [ffmpeg](https://ffmpeg.org/download.html)
* 安装Python模块：`pip install -r requirements.txt`

## 使用

1. 配置 `settings.py` 文件，设置原始视频路径 `video_path`
2. 运行 `run.py`，将会自动生成srt格式字幕文件至 `subtitles`文件夹
